Jupyterlab中显示Plotly的图形

Wuxf 发布于 2024-06-04 收录于 学习笔记

需要借助 IPython.dispaly.HTML 对象,在 Jupyterlab 安装的时候就已经可用了,具体用法如下:

  from IPython.display import HTML
  import plotly.graph_objects as go

  # 准备数据
  x = [1, 2, 3, 4, 5]
  y = [1, 4, 9, 16, 25]

  # 创建线形图
  fig = go.Figure()
  fig.add_trace(go.Scatter(x=x, y=y, mode="lines"))
  fig.update_layout(
      title="Simple Line Plot",
      xaxis_title="X",
      yaxis_title="Y",
      font=dict(family="Courier New, monospace", size=18, color="#7f7f7f"),
  )


  # 显示图形
  HTML(fig.to_html())

使用Easy hugo部署到github

作者 发布于 2023-10-03 收录于 Emacs

需要自己写一个 deploy.sh

  ##!/bin/bash

  # 将public目录中的所有文件提交到GitHub Pages的特定分支

  # 清除并重新生成Hugo站点
  hugo -d ./public

  # 进入public目录
  cd ./public

  # 初始化一个Git仓库并提交更改
  git init
  git add .
  git commit -m "Deploying to GitHub Pages"

  # 将更改推送到GitHub Pages的特定分支，例如"gh-pages"
  git push --force git@github.com:<username>/<username>.github.io.git master

  # 返回项目根目录
  cd ..

Debian系安装ping

Short description
June 14, 2024 · Wuxf

很多Docker镜像基于Debian或者ubuntu的精简版本，进入容器需要 ping 的时候经常找不到 ping ，需要安装，安装的命令如下：

1

  apt install iputils-ping

有时候还需要查看自己的代理是否生效，可以使用下面的命令：

1
2

  apt install curl
  curl cip.cc

cip.cc 是一个ip地址查询服务，也可以显示你当前的ip地址。
